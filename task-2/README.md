# User Guide
## Lightweight Event Hosting & Attendance Platform

This application helps communities run free events simply and efficiently.

**What you can do:**
- Hosts publish and manage events
- Attendees RSVP and receive digital tickets
- Staff check attendees in at the venue
- Communities share feedback and photos after events

---

## 1. Browsing Events (No Account Required)

Anyone can explore events without signing in.

### Explore Events
You can:
- Open the **Explore** page
- Browse upcoming and past events
- Search by event name or description
- Filter by date or location
- Open Event pages and Host pages

> ✅ No account is required to browse.

---

## 2. Creating an Account

You only need an account to take action (RSVP, host, or manage events).

### How to Sign In
1. Click **Sign In**
2. Enter your email address (use real email address to create an account, the confirmation email will be sent to it)
3. Complete the email sign‑in flow (confirm your email from the notification)

Once signed in, you can:
- RSVP to events
- View your tickets
- Become a Host
- Manage events (if you have a role)

---

## 3. Attending Events (Attendees)

### RSVPing to an Event
1. Open an upcoming Event page
2. Click **RSVP**
3. If you’re not signed in, you’ll be prompted to sign in first

**What happens next:**
- If spots are available → RSVP is **Confirmed**
- If the event is full → you’re added to the **Waitlist**

---

### Your Ticket
Immediately after RSVPing, you receive a digital ticket that includes:
- Event details
- A unique ticket code (used for entry)
- An **Add to Calendar** option

---

### My Tickets Page
The **My Tickets** page shows:
- All upcoming events you’ve RSVPed to
- Your current status (Confirmed or Waitlisted)

From this page, you can:
- Cancel your RSVP at any time

> ✅ Cancelled spots are automatically offered to people on the waitlist.

---

### Waitlist Promotion
If you’re on a waitlist:
- You’ll be automatically promoted when a spot opens
- Your status updates instantly in the app

No action is required from you.

---

## 4. Becoming a Host (Organisers)

Any signed‑in user can become a Host.

### How to Become a Host
1. Click **Become a Host**
2. Fill in your Host profile:
   - Name
   - Logo
   - Short bio
   - Contact email
3. Save your profile

You’ll get:
- A public **Host page**
- Access to event creation tools

---

## 5. Creating & Managing Events (Hosts)

### Creating an Event
From your Host area, click **Create Event** and provide:
- Event title and description
- Start and end date/time
- Venue address or online link
- Capacity
- Cover image
- Visibility:
  - **Public** (discoverable)
  - **Unlisted** (link‑only)

New events start in **Draft** mode.

---

### Event States
Events move through these states:
- **Draft** – visible only to you
- **Published** – visible to attendees
- **Unpublished** – hidden but saved

Available actions:
- Publish
- Unpublish
- Duplicate an event

---

### Free vs Paid Events
In the event editor you’ll see a **Free / Paid** toggle:
- Free events are fully supported
- Paid events are marked **Coming soon**

---

## 6. Discoverability & Sharing

- Public events appear on the Explore page
- Unlisted events are accessible via direct link only
- Event and Host pages generate clean previews for social sharing

---

## 7. Managing Attendance (Hosts)

### Host Dashboard
The dashboard shows:
- Upcoming events
- Past events
- Per-event statistics:
  - Confirmed attendees
  - Waitlisted attendees
  - Checked‑in attendees

---

### Exporting Attendee Data
Hosts can export a CSV for each event containing:
- Attendee name
- Email address
- RSVP status
- Check‑in timestamp

The CSV works in:
- Microsoft Excel
- Google Sheets

---

## 8. Team Members & Check‑In

### Roles
Each Host can assign roles:
- **Host** – full management access
- **Checker** – check‑in access only

### Inviting Team Members
Hosts invite members using a copyable link that assigns the chosen role.

---

### Checking In Attendees
On event day, Hosts or Checkers can open the **Check‑in** page.

Check‑in features:
- Manual ticket code entry (no camera needed)
- Instant validation
- Duplicate check‑ins are prevented
- Live counters for:
  - Checked‑in attendees
  - Remaining capacity
- Undo the last check‑in if needed

> ✅ Works well even with unstable internet.

---

## 9. After the Event

### Attendee Feedback
After an event ends, attendees can submit:
- A 1–5 star rating
- An optional comment

Each attendee can submit feedback once.

---

### Event Photo Gallery
- Attendees can upload event photos
- All uploads require Host approval
- Only approved photos appear publicly

---

## 10. Reporting & Moderation

Any user can report:
- An event
- A gallery photo

Hosts can:
- Review reported items
- Hide items from public view

---

## What This App Does Not Include (Yet)

To keep things simple, the app does **not** include:
- Payments or paid ticketing
- Email notifications
- Advanced analytics
- Mobile apps
- Camera‑based QR scanning

These features may be added in future versions.

---

## Summary

**Attendees can:**
- Discover events
- RSVP and manage tickets
- Attend smoothly and leave feedback

**Hosts can:**
- Create and promote events
- Manage attendance and check‑ins
- Export attendee data

**Checkers can:**
- Validate tickets quickly at the door

---

*This platform is designed to stay simple, friendly, and reliable — perfect for community‑driven events.*
